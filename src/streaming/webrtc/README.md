# streaming/webrtc

Server-side WebRTC streaming module for EdgeVision on Jetson Orin Nano. Publishes H.264 over `webrtcbin` at 854x480@30 / 1.5 Mbps with a single-client JSON signalling listener on a WebSocket port (default `:6689`). Defaults to TSDF-only (`enableRgb=false`); RGB goes via the existing TCP `RenderServer` to keep the encoder budget free. Orin Nano has no NVENC, so encoding uses software `x264enc` (`tune=zerolatency speed-preset=ultrafast threads=2`) pinned to a 2-core mask (default cores 4-5 via `pumpCoreMask`). Pose updates flow back from the client over a DataChannel and drive the TSDF raycast pose for the next frame. Construct `WebRtcServer` once at boot with the `FrameStore` and the `SharedScene`, call `start()`. Wire the TSDF raycaster via `setTsdfRenderer(closure)` once it's ready — until then the TSDF track stays silent (no black-frame overhead).
